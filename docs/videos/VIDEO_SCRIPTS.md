## Recording Instructions

### Technical Requirements

**Screen Resolution:** 1920x1080 (1080p) or 1280x720 (720p)  
**Frame Rate:** 30 fps  
**Audio:**
- Clear microphone
- Noise reduction
- Consistent volume level

**Recording Software Options:**
- OBS Studio (free, open-source)
- Playwright (automated with `npx playwright test --headed --video=on`)
- QuickTime Player (Mac)
- Windows Game Bar (Windows)
- ScreenFlow (Mac, paid)
- Camtasia (cross-platform, paid)

### Post-Production

**Editing Steps:**
1. Add on-screen text overlays at specified timestamps
2. Add subtle background music (optional, low volume)
3. Normalize audio levels
4. Add end screens with documentation links
5. Export in multiple formats:
   - MP4 (H.264, 1080p) - Primary
   - WebM (VP9) - Web optimization
   - GIF (for quick previews) - Optional

**Subtitles:**
- Generate .srt subtitle file from narration script
- Embed in video or provide separately
- Include timing markers for each subtitle block

### File Naming Convention

```
dashboard-[name]-[version]-[resolution].mp4

Examples:
- dashboard-walkthrough-v1-1080p.mp4
- dashboard-use-cases-v1-720p.mp4
- dashboard-status-system-v1-1080p.mp4
```

### Storage Location

## Playwright Automated Recording

### Setup

```bash
# Install Playwright
npm install -D @playwright/test

# Configure Playwright for video recording
npx playwright codegen --target javascript --save-storage=auth.json http://localhost:3000
```

### Test Script for Automated Recording

Create `tests/dashboard-demo.spec.ts`:

```typescript
import { test, expect } from '@playwright/test';

test('Dashboard Walkthrough', async ({ page }) => {
  // Navigate to Dashboard
  await page.goto('http://localhost:3000');
  await page.click('#nav-dashboard');
  
  // Wait for Dashboard to load
  await expect(page.locator('h1')).toContainText('Publisher Configuration Tool');
  
  // Hover over Create New Publisher
  await page.hover('button:has-text("Create New Publisher")');
  await page.waitForTimeout(1000);
  
  // Demonstrate search
  await page.fill('input[placeholder*="Search"]', 'aurora');
  await page.waitForTimeout(2000);
  
  // Clear search
  await page.fill('input[placeholder*="Search"]', '');
  
  // Click Active Only
  await page.click('button:has-text("Active Only")');
  await page.waitForTimeout(2000);
  
  // Clear All
  await page.click('button:has-text("Clear All")');
  await page.waitForTimeout(1000);
  
  // Click on a publisher card
  await page.click('text=Aurora Media');
});
```

### Run with Video Recording

```bash
# Run test with video
npx playwright test dashboard-demo.spec.ts --headed --video=on

# Videos saved to: test-results/*/video.webm
```
